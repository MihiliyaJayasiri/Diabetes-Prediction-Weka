# Diabetes Prediction Using WEKA

## Overview
This repository contains a university coursework project focused on predicting
whether a patient has diabetes using data mining techniques using the WEKA tool.

## Tools Used
- WEKA (Waikato Environment for Knowledge Analysis)
- Machine Learning classification algorithms

## Key Tasks
- Data preprocessing
- Model building using multiple algorithms
- Model evaluation and comparison
- Identification of the best-performing model

## Academic Context
This project was completed as part of the module
Principles of Data Science 1st Year 1st Sem (2024).
